# 🎓 Student Management System

A comprehensive console-based Student Management System developed in **C++** using **structured programming** and **file handling** concepts. The system provides separate Admin and Student panels for managing student records, calculating GPA, generating report cards, and tracking academic performance.

## 📖 Project Overview

This application is designed to simplify student record management in educational institutions. It allows administrators to perform CRUD operations, monitor academic performance, manage announcements, and generate report cards, while students can access their academic information.

All data is stored permanently using text files.

## ✨ Key Features

### 🔐 Admin Panel
- Secure password authentication
- Add new student records
- Update student information
- Delete student records
- Search students by ID
- Search students by Name
- View all students
- Generate report cards
- View class dashboard
- Sort students by ID or Name
- Generate GPA-based ranking
- Manage announcements
- Update student dues status
- View scholarship-eligible students

### 👨‍🎓 Student Panel
- Login using Student ID
- View personal information
- View subject-wise marks
- View GPA
- View academic status
- View performance

## 📊 Academic Features

- GPA calculation using a 4.0 grading scale
- Performance evaluation
- Pass/Fail determination
- Academic risk detection
- Class topper identification
- Average GPA calculation
- Pass/Fail statistics
